# System Architecture Diagram

## Prompt

```text
Draw a system architecture diagram for a Next.js SaaS app with Supabase auth, Postgres, Cloudflare R2 storage, Stripe billing, and Vercel deployment.
```

## What to Expect

The diagram should show:

- User/browser entry point
- Next.js app running on Vercel
- Supabase Auth for authentication
- Postgres database for application data
- Cloudflare R2 for uploaded files
- Stripe for billing and webhooks
- Arrows showing request flow and webhook callbacks

## Why This Demo Works

This is a familiar architecture for indie SaaS and developer tools. It is concrete enough for an AI agent to draw a useful first draft, but still broad enough to show that CodePic can handle real system diagrams.
