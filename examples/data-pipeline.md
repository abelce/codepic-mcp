# Data Pipeline Diagram

## Prompt

```text
Draw a data pipeline from Segment to Snowflake to dbt to Metabase, with batch jobs, transformation layer, and dashboard output.
```

## What to Expect

The diagram should show:

- Event collection from Segment
- Raw data landing in Snowflake
- Scheduled transformation jobs
- dbt models and transformation layer
- Clean analytics tables
- Metabase dashboards or reporting output

## Why This Demo Works

Data pipelines are easy to describe as a list of tools, but teams need to see direction, ownership, and transformation stages. This prompt demonstrates how MCP can produce a first draft of a pipeline diagram for documentation.

## Community Short Description

CodePic MCP can generate an editable analytics pipeline diagram from a short stack description: Segment → Snowflake → dbt → Metabase.
