# CI/CD Pipeline Diagram

## Prompt

```text
Create a CI/CD pipeline diagram for GitHub Actions deploying a Next.js app to Vercel with type-check, lint, build, preview deployment, and production deployment.
```

## What to Expect

The diagram should show:

- Pull request trigger
- Type-check job
- Lint job
- Build job
- Preview deployment to Vercel
- Main branch merge
- Production deployment
- Optional rollback or notification step

## Why This Demo Works

CI/CD diagrams are often described in README files, onboarding docs, and architecture reviews. Turning a deployment workflow into a visual diagram helps new team members understand the release path quickly.
